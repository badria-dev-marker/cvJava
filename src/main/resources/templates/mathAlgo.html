Triangle de Pascal

le triangle de Pascal est une présentation des coefficients binomiaux dans un triangle. Il fut nommé ainsi en l'honneur du mathématicien français Blaise Pascal. Les nombres obtenus sont en fait les coefficients du développement de: ( a + b )n

La notation de C nk désigne le nombre de parties de cardinal k dans un ensemble de cardinal n, autrement dit, C nk est le nombre façons de choisir k éléments parmi n éléments distincts (0 ≤ k ≤ n), aucun ordre n'intervenant dans ce choix.
